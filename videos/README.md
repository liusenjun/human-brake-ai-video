# 视频文件说明 / Videos

本目录仅存放视频文件的**链接**，不直接上传视频文件。

原因：AI 生成的视频文件较大（通常数十 MB 到数百 MB），直接上传 GitHub 会导致仓库体积过大，且免费账户有存储限制。

## 如何添加视频链接

1. 将视频上传到你偏好的云存储（百度网盘、Google Drive、OneDrive 等）
2. 生成分享链接
3. 在上表中添加链接（点击链接列的 `[待添加]` 替换为实际链接）
4. 通过 Pull Request 提交更新
