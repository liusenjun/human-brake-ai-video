3. CLIPTextEncode → positive prompt
4. CLIPTextEncode → negative prompt
5. LTXVConditioning → 连接 positive/negative clip
6. LoadImage → SHOT\shot1.png
7. EmptyLTXVLatentVideo → 1280×720×193 frames
8. LTXVImgToVideoConditionOnly → **vae=[1,2]**, image=[6,0], latent=[7,0], strength=1.0
9. LTXVTiledVAEDecode → **vae=[1,2]**, latents=[8,0], tiles=2×2, overlap=6
10. CreateVideo → fps=24
11. SaveVideo → prefix="shot1_ltx23_v3"
